# ⏰ Digital Clock Web Application

## 📌 Project Overview

The Digital Clock Web Application is a beginner-friendly front-end project developed using HTML, CSS, and JavaScript. The application displays the current system time and date in real-time and updates automatically every second without requiring a page refresh.

This project demonstrates fundamental web development concepts such as DOM Manipulation, JavaScript Date Object, Event Scheduling using setInterval(), and Responsive UI Design.

## 🧠 How It Works

1. JavaScript creates a Date object.
2. Current time is extracted.
3. Time is converted into 12-hour format.
4. AM/PM is determined.
5. Clock is updated on the webpage.
6. setInterval() refreshes the clock every second.

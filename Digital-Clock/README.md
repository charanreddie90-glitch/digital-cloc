# Digital Clock

## Project Description

Digital Clock is a web application that displays the current time and date in real time.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- Real-time clock
- Automatic updates every second
- AM/PM format
- Date display
- Responsive design

## How to Run

1. Download project.
2. Open project folder.
3. Double click index.html.
